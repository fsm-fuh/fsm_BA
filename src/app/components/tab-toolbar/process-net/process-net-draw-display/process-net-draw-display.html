Zeichenbereich

Ziehen Sie Elemente aus dem linken Anzeigebereich hierher, um Ihr Prozessnetz zu erstellen.

Mit Linksklick Elemente ziehen, um sie zu bewegen.

Mit Mittelklick einen Place oder eine Transition löschen (Verbindungen werden automatisch entfernt).

Rechtsklick auf einen Place oder eine Transition, um ihn als Quelle zu wählen; anschließend Rechtsklick auf den gegen gesetzten Typ, um eine gerichtete Verbindung zum zweiten Element zu erstellen.

Mittelklick auf eine Verbindung, um sie zu löschen.

Validieren, um eine Toast‑Nachricht mit dem Ergebnis zu sehen.
